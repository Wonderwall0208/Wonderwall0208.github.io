Encrypt a Note

🔒 Everything runs locally — the file and password never leave this page.

Pick a PDF and a password. You'll get an encrypted .enc file to put in the notes/ folder. Only someone with the password can open it on the Notes page.

Click to choose a PDF
or drag & drop it here

Then

  1. Move the downloaded .enc into notes/.
  2. In notes.html, add a list item pointing data-enc to it.
  3. Do not upload the original .pdf — only the .enc.
